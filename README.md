# Kalkulator Bunga Sederhana

Proyek kalkulator bunga sederhana untuk tugas akhir Coursera.

## Deskripsi
Skrip ini menghitung bunga sederhana berdasarkan pokok pinjaman, suku bunga tahunan, dan periode waktu dalam tahun.

## Parameter Input
* **p**, jumlah pokok pinjaman
* **t**, periode waktu dalam tahun
* **r**, suku bunga tahunan

## Output yang Diharapkan
Skrip akan menampilkan hasil perhitungan bunga sederhana menggunakan rumus: `p * t * r / 100`.

## Contoh Penggunaan
```bash
./simple-interest.sh
Masukkan pokok:
1000
Masukkan suku bunga tahunan:
5
Masukkan periode waktu dalam tahun:
2
Bunga sederhana adalah: 
100
